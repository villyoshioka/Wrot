# 装飾の使い方

Wrot の投稿フォームには、書式を整えるためのツールバーが付いています。
ボタンを押すだけで太字や引用、リストなどの Markdown を挿入できます。
このページでは、ツールバーから使える装飾と、その振る舞いをまとめています。

---

## ツールバーの全体像

投稿フォームのテキストエリアの下に、左から順に以下のボタンが並んでいます。

| アイコン | 名前               | 概要                                                  |
| -------- | ------------------ | ----------------------------------------------------- |
| 🖼️       | 画像追加           | 端末のファイル選択ダイアログを開いて画像を添付       |
| 📎       | 埋め込み           | `![[ ]]` でノートやファイルを埋め込み                |
| **B**    | 太字               | `**強調**`                                            |
| *I*      | 斜体               | `*斜体*`                                              |
| ☰        | リスト             | `- 項目`                                              |
| ☑        | チェックリスト     | `- [ ] タスク`                                        |
| 1.       | 番号付きリスト     | `1. 項目`                                             |
| ⋯        | その他の書式メニュー | コード・数式・引用・リンク・取り消し線・ハイライト |

---

## 太字・斜体

選択範囲があるときと、ないときで振る舞いが変わります。

### 選択範囲があるとき

選んだ部分をそのまま装飾します。もう一度押すと装飾が外れます。

- 太字: 選んだ文字を `**…**` で囲みます
- 斜体: 選んだ文字を `*…*` で囲みます

### 選択範囲がないとき（予告モード）

カーソル位置に**マーカーだけ先に挿入**して、書きながら装飾を確定する仕組みです。

1. 太字ボタンを押すと、カーソル位置に `**` が挿入されます（ボタンが点灯）
2. そのまま書きたい文字を入力します
3. もう一度太字ボタンを押すと、閉じの `**` が入って装飾が完成します

斜体も同じ要領です。

### 太字と斜体は同時に使えない

「太字を予告中」のときに斜体ボタンを押しても反応しません。逆も同じです。
書き終えて閉じてから、もう一方を使ってください。

### IME で日本語を入力中の振る舞い

IME で文字を変換している間は、予告中のボタンが押せなくなります。
意図せず装飾を閉じてしまうのを防ぐためのガードです。

---

## リスト・チェックリスト・番号付きリスト

ボタンを押すと、**カーソル行の先頭**に印を挿入します。
押した位置が行の途中でも、挿入は必ず行頭に入ります。

| ボタン           | 挿入される印 |
| ---------------- | ------------ |
| リスト           | `- `         |
| チェックリスト   | `- [ ] `     |
| 番号付きリスト   | `1. `        |

### Enter キーで次の項目へ

リストの項目で改行すると、次の項目の印が自動で入ります。
空の項目で Enter を押すと、リストから抜けます。

---

## コード

「その他」メニューから「コード」を選びます。

| 状況             | 挿入される形                       |
| ---------------- | ---------------------------------- |
| 選択範囲あり     | <code>\`選択\`</code>（インライン） |
| 選択範囲なし     | <code>~~~</code> 〜 <code>~~~</code> （フェンスブロック）|

フェンスブロックの中にカーソルが置かれた状態になるので、そのまま中身を書けます。

---

## 数式

「その他」メニューから「数式」を選びます。
コードと同じく、選択範囲の有無で振る舞いが変わります。

| 状況             | 挿入される形              |
| ---------------- | ------------------------- |
| 選択範囲あり     | `$選択$`（インライン）    |
| 選択範囲なし     | `$$ 〜 $$`（ブロック）   |

---

## 引用

「その他」メニューから「引用」を選びます。
選んだ行（または現在の行）の先頭に `> ` をトグルで付け外しします。

- 複数行を選んでいると、すべての行に `> ` が付きます
- すべての行に `> ` が付いている状態で押すと、まとめて外れます

---

## リンク

「その他」メニューから「リンク」を選びます。
**選択範囲がないと使えません**（メニュー項目がグレーアウトします）。

選んだ文字を `[文字](URL)` の形に変換し、`URL` の位置にカーソルを置きます。

例: 「Wrot」を選んで押すと、`[Wrot]()` になり、`()` の中にカーソルが入った状態になります。

---

## 取り消し線・ハイライト

| ボタン     | 装飾    |
| ---------- | ------- |
| 取り消し線 | `~~…~~` |
| ハイライト | `==…==` |

どちらも「その他」メニューから選びます。
**選択範囲がないと使えません**（メニュー項目がグレーアウトします）。
もう一度押すと装飾が外れます。
