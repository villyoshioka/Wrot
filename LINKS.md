# 画像・リンク・埋め込みの使い方

Wrot では Obsidian と同じ記法で、画像・ノートへのリンク・URL を扱えます。
このページでは、それぞれの書き方と表示のされかたをまとめています。

---

## 📎 埋め込みボタン

入力欄ツールバー先頭の 📎 ボタンは、Wrot で扱えるあらゆる埋め込みの入り口です。
**画像も、Obsidian のノートも、このボタン 1 つで貼れます。**

### 使い方

1. 📎 を押すと `![[ ]]` が挿入され、カーソルが内側に置かれます。
2. その状態でファイル名を入力すると、Obsidian のファイル名サジェストが出ます。
3. サジェストから選ぶと完成。画像なら画像、ノートならリンクとして表示されます。

### 選択したテキストを囲みたいとき

- ファイル名を選択してから 📎 を押すと、`![[ ]]` で自動的に囲まれます。
- すでに `![[ ]]` または `[[ ]]` で囲まれた範囲を選択して 📎 を押すと、解除できます。

### カーソルが `![[ ]]` の中にあるときの挙動

- 📎 ボタンが「アクティブ状態」で点灯します。
- もう一度押すと `![[ ]]` を解除できます。

---

## 🖼️ 画像の貼りかた

Wrot で画像を表示する方法は 3 つあります。すべて 📎 ボタンや手書きで書けます。

### 1. Vault 内の画像（おすすめ）

```
![[photo.png]]
```

→ Vault 内に画像があれば、そのまま画像として表示されます。

対応している拡張子: `png` `jpg` `jpeg` `gif` `svg` `webp` `bmp`

### 2. インターネット上の画像

```
https://example.com/photo.png
```

→ URL プレビュー設定が ON でも OFF でも画像として表示されます。

### 3. Obsidian URL での画像

```
obsidian://open?vault=MyVault&file=photo.png
```

→ Vault 内の画像が解決できれば、画像として表示されます。
　 解決できない場合はリンクとして表示されます。

---

## 🔗 ノート間リンク

Vault 内のノートへのリンクも貼れます。

```
![[ノート名]]
[[ノート名]]
```

📎 ボタンで挿入されるのは `![[ ]]` のみです。`[[ ]]` を使いたい場合は手書きしてください。
どちらの記法でも、クリックすると Obsidian がそのノートを開きます。
画像と違い、ノートはリンクとして表示されます。

---

## 🌐 URL プレビュー（OGP カード）

外部リンクは、リンク先の情報を OGP カードとして表示できます。

### カードが表示される条件

- 設定で「URL プレビュー」が ON になっている
- `http://` または `https://` の URL である
- リンク先からタイトルまたは説明文が取得できる

### カードが表示されないケース

- 設定で「URL プレビュー」が OFF
- `obsidian://` URL（Vault 内のファイルなので OGP 情報がない）
- 画像 URL（カードではなく画像プレビューとして表示される）
- リンク先からタイトル・説明文が取得できなかった

### Twitter / X の投稿

`https://twitter.com/...` や `https://x.com/...` の投稿リンクは、専用のスタイルで表示されます。

---

## 困ったとき

### 画像が表示されない

- ファイル名と Vault 内のファイルが一致しているか確認してください
- 拡張子が対応しているか確認してください
- `![[ ]]` の括弧が正しく閉じているか確認してください

### URL プレビューが出ない

- 設定で「URL プレビュー」を ON にしてください
- リンク先が OGP 情報を持っていない場合は表示されません
- 画像 URL の場合は OGP カードではなく画像として表示されます
